# Сообщение

Программам нужно уметь передавать информацию главной программе - для этого я создал шаблон:
```json
{
  "source": "berserk-checker",
  "status": "ok",
  "payload": {
    "error_code": 1,
    "message": "Couldn't reach the API."
  }
}
```
Внутри этого шаблона, payload - это суть передаваемой информации от подпрограммы к программе-получателю. 

Всё что вне payload, то есть, "source", "status", присутствовать должно __обязательно__.

## Сервер

Было несколько вариантов осуществить это, но самый лучший (и после - масштабируемый) является локальный сервер. 

Это не просто удобно, но и может выйти за пределы одного устройства, создавая единую систему внутри сети.
